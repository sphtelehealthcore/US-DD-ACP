## Overview

The purpose of this repository is to provide transparency and reproducibility for the statistical analyses presented in the manuscript. The repository includes:
- Packages and functions
- Data cleaning scripts
- Statistical analysis code
- Figure and table generation scripts
- Documentation for reproducing the analytical workflow
